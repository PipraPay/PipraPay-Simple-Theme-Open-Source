== Description ==

**PipraPay Vercel** is a modern, responsive, and lightning-fast theme tailored for PipraPay environments. It provides a clean and user-friendly UI for managing payments, invoices, transactions, and SMS notifications.

**Key Features:**
* Fully responsive layout – looks great on desktop and mobile
* Optimized for speed and performance
* Designed specifically for PipraPay payment systems
* Seamless integration with PipraPay SMS and transaction alerts
* Built with modern web standards (HTML5, CSS3, Flexbox)
* Lightweight and minimal – no bloat
* Developer-friendly structure
* Easy to customize and extend
